# Vidi Vouch — website

The public landing page for **Vidi Vouch**, a code-review provenance CLI.
Served via GitHub Pages.

## Develop

A single self-contained `index.html` (inline CSS + SVG, no build step, no
external assets). Open it in a browser to preview, then edit, commit, push.
Pages rebuilds from `main` on every push.
